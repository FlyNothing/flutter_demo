# 1 项目介绍

- 项目名称：Flutter Demo项目
- 项目功能：Flutter开发过程的基本Widget功能和Flutter特性、扩展插件的使用及注意事项、常见问题的修复

# 2 目录说明

## 2.1 整体目录

| 目录      | 说明             |
| ------- | -------------- |
| android | android平台相关代码  |
| asset   | 项目资源，如配置文件、图片等 |
| ios     | ios平台相关代码      |
| lib     | 项目Main代码       |
| test    | 项目Test代码       |
| web     | web平台相关代码      |

## 2.2 lib目录

| 目录     | 说明                   |
| ------ | -------------------- |
| common | 通用模块                 |
| base   | 基本Widget功能和Flutter特性 |
| issue  | 常见问题及处理              |
| plugin | 扩展插件                 |

# 3 更新说明

当前项目主要更新分为2块：

- base：更新一些Flutter自带的部件、功能及插件的使用

- plugin：pub.dev中一些使用较多的第三方或Flutter官方扩展插件的使用和介绍
